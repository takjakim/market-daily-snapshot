# market-daily-snapshot

Daily market snapshot script (indices + top movers) intended for personal automation.

## Files
- `daily_market_prices.py`
- `requirements-daily-prices.txt`

## Usage
```bash
python3 -m pip install --user -r requirements-daily-prices.txt
python3 daily_market_prices.py
```

Notes:
- Uses free public endpoints (best-effort). Rate limits can occur.
